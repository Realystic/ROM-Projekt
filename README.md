# Videoigre do leta 2024

Pri projektu bom uporabil racunalniski program *Mathematica*.

## Načrt Projekta
1. Pridobiti podatke o videoigrah in njihove lastnosti
2. Razvrstili te podtatke in jih prikazali z grafi in tabelami
